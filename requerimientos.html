Requerimientos

Requerimientos funcionales

- Presentar información corporativa (historia, trayectoria, misión, visión, valores)

- Mostrar perfiles generales de los abogados y profesionales

- Incluir datos de ubicación, contactos, alianzas y reconocimientos.

- Formulario para registro de hoja de vida.

- validación y almacenamiento de datos de candidatos.

-Funcionalidad para evaluar y aceptar / rechazar postulantes.

- Registro de clientes

-Panel para que cada profesional gestione sus casos.

- Control de usuarios y roles (Administrador, Abogado, candidato, cliente)

-generación de reportes sobre procesos de candidatos y clientes.

Requerimientos no funcionales

- Diseño responsive para que pueda acceder de manera amigable a cualquier dispositivo.

- La interfaz debe ser intuitiva y fácil de usar para que todos los actores, con un diseño profesional y acorde a la imagen corporativa.

- autenticación de usuarios, control de acceso de roles, cifrado de datos sensibles.

- La pagina de soportar el crecimiento en cantidad de usuarios, servicios y datos.

- Copias de seguridad automáticas para evitar perdida de información.

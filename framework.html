Frameworks

1. Bootstrap: Es un framework CSS y JS que incluye un sistema de rejillas responsive, componentes preconstruidos y utilidades para crear interfaces estables. Es utilizado mayormente para proyectos que se desarrollen de forma rápida, ya que este permite una facilidad de menus, formularios, botones y otros elementes con estilos predeterminados.

2. Tailwind CSS: Es un framework que permite crear interfaces personalizadas sin que estos dependan de estilos ya predefinidos. Utiliza clases pequeñas y específicas, lo que ofrece una mejor flexibilidad para el diseño de la plataforma.

3. Foundation: Es desarrollado por Zurb, y es un framework frontend avanzado que ofrece sistemas de rejillas adaptables, componentes accesibles y herramientas para crear interfaces modernas. Esta destaca por su accesibilidad lo cual ayuda a las personas con discapacidades.

4. Wordpress: Es un gestor de contenido (CMS) que permite crear paginas web de una forma rápida sin la necesidad de programar desde cero. Con un puglin de diseño visual llamado Elementor, es posible crear paginas responsive arrastrando y soltando componentes.

5. Angular: Es un framework de desarrollo frontend que es creado por Google que utiliza TypeScript. Esta pensado para construir pagians web de una sola página robustas y escalables. Esta proporciona una arquitectura basada en componentes y herramientas integradas como enrutamiento, manejo de formularios, validaciones, inyección de dependencias y comunicación con Apis.

6. React: Es una librearía JavaScript creado por Meta, para construir interfaces de usuario dinámicas. Utiliza el concepto de componentes reutilizables y el Virtual DOM, lo que hace mejorar el rendimiento y permite una experiencia de usuario más fluida.
